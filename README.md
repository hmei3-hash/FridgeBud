# FridgeBud

git push test

Smart Fridge Companion that tracks food freshness and reduces waste using AI vision, sensors, and a touchscreen interface.

## Overview

FridgeBud is an IoT system that attaches to a refrigerator and helps users manage food inventory automatically.  
Using an internal camera, AI food recognition, and a touchscreen interface, FridgeBud detects what food is stored, tracks expiration dates, and reminds users before food goes bad.

The goal is to **reduce food waste, improve food organization, and simplify meal planning**.

## Key Features

- AI food recognition using computer vision
- Automatic expiration tracking
- Touchscreen interface mounted on fridge
- Manual food entry option
- Food freshness alerts
- Shopping suggestions based on inventory
- Cloud sync and mobile notification (future)

## System Architecture

FridgeBud combines hardware, embedded systems, and AI software.
